# coti-web3.py

Welcome to COTI Web3.py, a powerful extension to the popular [web3.py]() library, designed to unlock the full potential of the COTI blockchain network. This package provides developers with seamless access to all COTI features, extending the standard Web3 functionality to meet the unique requirements of building decentralized applications (dApps) and smart contracts on the COTI ecosystem.

## Installation

```bash
pip install ".[dev]"
```

## Testing

```bash
pytest
```
